# ShareButton Component

A reusable **Share Button UI component** for a portfolio built with **React + Vite + Tailwind CSS + Framer Motion**.

---

## 📌 Overview

The `ShareButton` component provides a clean and modern way to share your portfolio. It is positioned in the **top-right corner** and includes smooth animations, theme support, and social sharing options.

---

## ⚙️ Features

### 🔘 Button UI

* Circular button with soft shadow
* Fixed position (top-right corner)
* Minimal, modern design

### 🎬 Animations (Framer Motion)

* Hover: `scale: 1.1`
* Click: `scale: 0.95`
* Popup: smooth slide + fade animation

### 🎨 Styling

* Light Theme: white background with slate border
* Dark Theme: slate-950 background with white border
* Cyan accent color on hover
* Consistent spacing using Tailwind scale
* Soft shadows for depth

### 🧠 Behavior

* Click to toggle share menu
* Click outside to close automatically
* Smooth transitions for better UX

---

## 📤 Share Options

### 1. Copy Link

* Copies portfolio URL to clipboard
* Shows confirmation feedback

### 2. WhatsApp

* Opens WhatsApp with pre-filled message

### 3. LinkedIn

* Opens LinkedIn share dialog

### 4. Twitter / X

* Opens Twitter with pre-filled tweet

---

## 📱 Responsiveness

* Works across all screen sizes
* Fixed positioning ensures accessibility on mobile and desktop

---

## 🧩 Design System

* Colors: cyan accents + slate neutrals
* Typography: clean and professional hierarchy
* Icons: interactive with hover scaling

---

## 🌗 Theme Support

* Fully compatible with light and dark modes
* Automatically adapts to your portfolio theme

---

## 📁 File Structure

```
src/
  components/
    ShareButton.tsx
```

---

## 🚀 Usage

Import and use inside your layout or main page:

```tsx
import ShareButton from "./components/ShareButton";

function App() {
  return (
    <div>
      <ShareButton />
    </div>
  );
}
```
