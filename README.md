# Nisha Portfolio React

A modern responsive portfolio web application built with React, Vite, and Tailwind CSS.

## Features
- Clean, minimal, professional UI
- Soft color palette with glassmorphism styling
- Dark/Light mode toggle
- Smooth scroll navigation and hover animations
- Sticky navbar and responsive layout
- Hero, About, Skills, Experience, Projects, Contact, and Footer sections

## Tech Stack
- React
- Vite
- TypeScript
- Tailwind CSS
- React Icons

## Setup
```bash
cd portfolio-react-vite
npm install
npm run dev
```

## Build
```bash
npm run build
npm run preview
```
