StreamExample.java — Java 8 Streams + Lambdas
Java 8+Streams
import java.util.*;
import java.util.stream.*;

List<Integer> nums = Arrays.asList(1, 2, 3, 4, 5, 6);

// Filter even numbers, square them, collect to list
List<Integer> result = nums.stream()
    .filter(n -> n % 2 == 0)          // 2, 4, 6
    .map(n -> n * n)                    // 4, 16, 36
    .collect(Collectors.toList());

// Group students by grade
Map<String, List<Student>> byGrade = students.stream()
    .collect(Collectors.groupingBy(Student::getGrade));

// Optional — safe null handling
Optional<String> name = Optional.ofNullable(getName());
String result2 = name.orElse("Unknown");

// Method references
nums.forEach(System.out::println);

UserController.java — Spring Boot REST API
Spring BootREST
@RestController
@RequestMapping("/api/users")
public class UserController {

    @Autowired
    private UserService userService;

    @GetMapping("/{id}")
    public ResponseEntity<UserDTO> getUser(@PathVariable Long id) {
        UserDTO user = userService.findById(id);
        return ResponseEntity.ok(user);  // 200 OK
    }

    @PostMapping
    public ResponseEntity<UserDTO> createUser(
            @Valid @RequestBody CreateUserRequest req) {
        UserDTO created = userService.create(req);
        return ResponseEntity
            .status(HttpStatus.CREATED)  // 201
            .body(created);
    }
}

1
What happens inside HashMap?
hashCode() called → bucket index → LinkedList for collision → Red-Black Tree at 8+ entries → O(1) avg, O(log n) worst
2
Difference between ArrayList and LinkedList?
ArrayList: O(1) get, O(n) insert/delete LinkedList: O(n) get, O(1) insert/delete at pointer Use ArrayList for random access, LinkedList for frequent insert/delete

Do NOT Skip These
What Most People Miss
❌ Not Writing Clean Code
Naming, formatting, single responsibility — interviewers judge code quality, not just correctness. Practice writing readable code always.
❌ Skipping Debugging Skills
Knowing IntelliJ debugger, reading stack traces, and fixing NullPointerException efficiently is a core real-world skill.
❌ Not Building Real Projects
Tutorial code ≠ project experience. Build a full CRUD REST API, add JWT, deploy it. That's what impresses interviewers.
❌ Memorizing Syntax, Not Understanding
Know WHY HashMap uses hashing, WHY threads need synchronization. Interviewers dig deeper — understand the "why".
